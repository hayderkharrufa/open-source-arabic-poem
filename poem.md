**ضحكت عُبيلة إذ رأت أكوادي**
<br/>
**مهلًا، فداكِ طارفي وتلادي**
<br/>
<br/>
**لا تحكمي قبلَ التبينَ إنني**
<br/>
**أدرى الأنامُ بمنطقِ الأعدادِ**
<br/>
<br/>
**يا ربَّ كودٍ قد كتبتُ سُطوره**
<br/>
**في ظهرِ لوحٍ مِن زمانِ إيادِ**[^1]
<br/>
<br/>
**حتّى إذا لقمتُهُ في آلةٍ**
<br/>
**كانَ الفتى في والصِّدرِ والإيرادِ**
<br/>
<br/>
**يا عبلُ كم من ليلةٍ قضيتُها**
<br/>
**في وسطِ معمةٍ من الأضدادِ**
<br/>
<br/>
**يا عبلُ كودي من حِجاي نسجتُه**
<br/>
**غَيري حِجاهُ ڤايب الكوادي**[^2]
<br/>
<br/>
**ونظام ويبٍ قد بنيتُ وموقع**
<br/>
**كـ"القصرِ ذي الشرفاتِ من سندادِ"**
